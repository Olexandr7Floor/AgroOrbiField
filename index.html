How to Use GIS Tools

To run the simulation, you need to **draw** the boundaries of your agricultural field on the map using the Leaflet Draw tools.

  1. Select the Drawing Tool: In the map toolbar (usually top left), click the **polygon** icon ().
  2. Draw the Field: Click on the map to set the corners of your field. To complete the polygon, click on the starting point.
  3. Check Data: After drawing is complete, the field **Area** will appear below the map, and the **centroid** (middle point) coordinates will update in the sidebar.
  4. Editing: If you need to change the shape, click the **edit** icon () on the toolbar.
  5. Change Layer Date: Use the **Layer Date** selector (usually bottom left of the map) to change the date of satellite layers (NDVI, LST, True Color). This is important for viewing data from a specific time.
  6. Run Simulation: Once the field is drawn and parameters (Crop, Irrigation, Fertilizer) are selected, the **"Run Simulation"** button will become active.
